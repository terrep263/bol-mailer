# BOL Mailer — Coolify Deployment Guide

## Step 1 — Create GitHub Repo
- Go to github.com/terrep263
- Create new repo named: bol-mailer
- Set to private
- Do NOT initialize with README

## Step 2 — Push Code from Local
Open terminal in G:\bol-mailer and run:
```
git init
git add .
git commit -m "initial: bol-mailer AI email automation app"
git remote add origin https://github.com/terrep263/bol-mailer.git
git branch -M main
git push -u origin main
```

## Step 3 — Deploy in Coolify
1. Log into Coolify at http://2.24.195.19:8000
2. New Resource → Application → GitHub → terrep263/bol-mailer → branch: main
3. Set Build Pack: Dockerfile
4. Set FQDN: mailer.thebookoflies.online
5. Add these Environment Variables:
   - ANTHROPIC_API_KEY = (your Anthropic API key)
   - LISTMONK_URL = https://mail.shopmyneighborhood.com
   - LISTMONK_USER = terrep263
   - LISTMONK_PASSWORD = 3020Herold!@!
   - CRON_SECRET = (generate a random 32-character string)
   - NEXT_PUBLIC_SITE_URL = https://mailer.thebookoflies.online
6. Click Deploy

## Step 4 — Initialize Listmonk Lists
After deploy is healthy, run this once:
```
curl -X POST https://mailer.thebookoflies.online/api/lists/setup
```
This creates the BOL Faith Pre-Launch list in Listmonk automatically.

## Step 5 — Set Up Cron Job in Coolify
1. In Coolify → your server → Scheduled Tasks (or use a cron container)
2. Schedule: 0 8 * * * (every day at 8am)
3. Command:
```
curl -H "x-cron-secret: YOUR_CRON_SECRET" https://mailer.thebookoflies.online/api/cron
```
Replace YOUR_CRON_SECRET with the value you set in Step 3.

## Step 6 — Install WordPress Plugin
1. Copy wordpress-plugin/bol-optin.php to your WordPress site:
   /wp-content/plugins/bol-optin/bol-optin.php
2. Activate the plugin in WordPress Admin → Plugins
3. Go to Settings → BOL Mailer → confirm URL is https://mailer.thebookoflies.online
4. Add shortcode [bol_optin] to any page or post on thebookoflies.online

## Step 7 — Set Up Listmonk Transactional Template
1. Log into Listmonk at https://mail.shopmyneighborhood.com
2. Go to Templates → New Template
3. Name: Default Transactional
4. Type: Transactional
5. Body:
```
{{ .Data.body }}
```
6. Save — this becomes Template ID 1

## Step 8 — Configure Listmonk SMTP
1. In Listmonk → Settings → SMTP
2. Add your SMTP credentials for hello@thebookoflies.online
3. Test and save

## Done
Once all steps are complete:
- Someone opts in on thebookoflies.online
- Email 1 is generated by Claude and sent immediately via Listmonk
- Every morning at 8am the cron checks who needs the next email
- Claude generates a unique email per subscriber per step
- Sequence completes automatically after 5 emails
